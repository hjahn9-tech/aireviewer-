# CREX AI Reviewer — 에이전트 설계 문서

## 멀티에이전트 아키텍처

```
사용자 입력 (시나리오)
        ↓
[persona_orchestrator] — 오케스트레이터
        ↓ 순차 호출
  ┌─────────────────────────────────────┐
  │ 1. persona_fetcher                  │
  │    HuggingFace API → 20개 원시 페르소나 │
  └────────────────┬────────────────────┘
                   ↓
  ┌─────────────────────────────────────┐
  │ 2. persona_builder                  │
  │    원시 데이터 → CREX 5차원 온톨로지    │
  └────────────────┬────────────────────┘
                   ↓
  ┌─────────────────────────────────────┐
  │ 3. persona_evaluator                │
  │    20개 페르소나 독립 평가 → 토론 → 등급 │
  └────────────────┬────────────────────┘
                   ↓
         최종 리포트 (JSON + Markdown)
```

## 스킬 연동

| 에이전트 | 사용 스킬 |
|---------|---------|
| persona_builder | persona-ontology-builder |
| persona_evaluator | scenario-evaluator |
| persona_evaluator | debate-facilitator |

## 5차원 평가 체계

| 차원 | 이름 | 설명 |
|-----|-----|-----|
| A | 취향 공동체 적합도 | Netflix 취향 클러스터 기반 장르 정합성 |
| B | 서사 구조 | 기승전결/비선형/에피소드 구조 선호도 매핑 |
| C | FRBR 요소 민감도 | 캐릭터/사건/배경/개념 민감도 |
| D | 몰입도 유형 | 완주형/클립형/정주행형 예측 |
| E | 틈새 취향 지수 | 장르 조합 수용도 및 시장 규모 |

## 종합 등급 기준

| 등급 | 기준 | 의미 |
|-----|-----|-----|
| GO | 평균 7.5 이상 | 상업적 출시 권장 |
| CAUTION | 5.0~7.4 | 보완 후 출시 검토 |
| PASS | 5.0 미만 | 전면 재검토 필요 |
