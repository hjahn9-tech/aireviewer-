# Persona_Agent — 오케스트레이터

## 역할

사용자로부터 시나리오를 입력받아, 3개의 서브에이전트를 순차적으로 조율하고 최종 평가 리포트를 생성한다.

## 실행 흐름

1. `persona_fetcher`를 호출하여 Nemotron-Personas-Korea에서 20개 페르소나를 무작위 수집
2. `persona_builder`를 호출하여 수집된 페르소나를 관객 온톨로지 형식으로 변환
3. `persona_evaluator`를 호출하여 관객 페르소나 간 토론 및 점수 산출
4. 최종 리포트(JSON + 마크다운)를 반환

## 출력 형식

- 페르소나별 점수 (A~E 5차원)
- 의견 충돌 페르소나 페어 목록
- 종합 추천 등급 (GO / CAUTION / PASS)
- 취향 도플갱어 연결 작품

## 오류 처리

- persona_fetcher 실패 시: 캐시된 페르소나 사용 또는 재시도 최대 3회
- 평가 타임아웃 시: 부분 결과로 리포트 생성 후 사용자에게 알림
- API 한도 초과 시: 사용자에게 명확히 안내 후 중단
