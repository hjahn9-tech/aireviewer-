# Persona Ontology Builder Skill

## 변환 절차

1. 원시 프로파일에서 연령·성별·직업·거주지 추출
2. KOFIC 2026 트렌드 점수 배정 (SF/테크: 32%, 호러: 22%, 시대극: 15%)
3. Netflix 취향 공동체 클러스터 중 가장 가까운 1개 배정
4. 5차원 온톨로지 태그 생성

## 다양성 보정 규칙

- 20개 페르소나 중 연령대별 4개 이상 확보 (10대, 20대, 30대, 40대+)
- 성별 균형 유지 (±20% 이내)
- 수도권/비수도권 비율 6:4 유지
